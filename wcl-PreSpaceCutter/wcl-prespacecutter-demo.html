About WCL-PreSpaceCutter

PreSpaceCutter is simple plugin which makes coding pre-formatted text easier.

When you are working on your html code and you have to write some pre-formatted text, it is a little bit problematic. In most cases you want to put your PRE text starting from left border of your container. If you want to archieve this you have to remove indentations from your code what obviously breaks code flow in your editor.

PreSpaceCutter will help you to solve this problem.

How to add WCL-PreSpaceCutter to your project?

Plugin is just JavaScript file (wcl-prespacecutter.js) - put it somewhere in your project's directory and link it at the end of body of your document: <script src="wcl-prespacecutter.js"> </script> - it will run automatically when your page is loaded.

How WCL-PreSpaceCutter works?

Basically WCL-PreSpaceCutter removes spaces from left side of your preformatted text but keeps line structure of this text. If the first or the last line of text is empty or spaces-only - it is also removed. That's all. What does it mean for you? You can write your pre-formatted text code according to your code indentations and plugin will remove unecessary spaces on-fly assuming that your leftmost line starts at the left border of the container.

Which elements are processed?

By default WCL-PreSpaceCutter processes CODE elements which are direct children of PRE elements ( pre > code) and all PRE elements. In most cases this is enough, but it's very easy to change this selection of elements beacause plugin starts with parameter which is simple jQuery selector. Take a look into source code of the plugin - it starts in first lines. If you want to change processed elements just change parameter in wclPreSpaceCutterRun() - according to jQuery selector rules. You can select elements with particular ID or class, but remember: this plugin checks type of element first - if element is CODE then plugin processes html code of this element, if element is not CODE then plugin processes all text nodes of the element.
